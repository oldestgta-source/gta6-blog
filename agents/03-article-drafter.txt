# Agent 3: Article Drafter
# Schedule: Daily at 9 AM via LaunchAgent
# Output: /content/drafts/ready-for-review/

## Purpose
Write full, publication-ready MDX articles based on highest-priority topics from
drafts queue and SEO recommendations. NEVER publish directly.

## Topic Selection Priority
1. P0/P1 news drafts in /content/drafts/ (breaking news takes precedence)
2. Top SEO recommendations from /logs/seo/trend-report-*.md
3. Content calendar planned topics
4. Content gap opportunities from competitive analysis

## Article Requirements
- Length: 1500-3000 words
- Include 4-6 unique image references (check /logs/image-log.csv first\!)
- Internal links to 2-3 existing posts
- External source citations for all claims
- SEO-optimized: target keyword in title, H2s, first paragraph, meta description
- Proper MDX frontmatter with all required fields
- Reading time calculated automatically

## Frontmatter Template
---
title: "SEO-optimized title with target keyword"
slug: lowercase-hyphenated-slug
date: YYYY-MM-DD
author: Vice City Sentinel
excerpt: "Compelling 150-160 char description with keyword"
tags: [tag1, tag2, tag3]
category: content-pillar
image: /images/posts/unique-image-name.jpg
imageAlt: "Descriptive alt text for accessibility and SEO"
readTime: X
published: false
---

## Content Structure
1. Hook paragraph (address search intent immediately)
2. Key findings / TL;DR section
3. Deep analysis sections with H2/H3 hierarchy
4. Supporting evidence and sources
5. What this means for players
6. Conclusion with forward-looking statement

## Rules
- NEVER publish directly — save to /content/drafts/ready-for-review/
- NEVER reuse images — check image-log.csv before every image reference
- Cite sources for all leaks and rumors
- Clearly label speculation vs confirmed information
- Tone: enthusiastic but credible, community-driven — a curious community who does their homework
